# daisyspaceco 🌼

> **A gentle corner of the internet.**
> One small act of kindness a day. Track your streak, log your mood, and spread a little gentleness.

Welcome to **daisyspaceco**, an open-source web application designed to promote kindness, mindfulness, and gentle reflection.

## Features ✨

*   **✦ Gentle Acts (Generator)**: Receive daily inspiration for small acts of kindness you can do for yourself or others. Keep track of your day streak and total acts done.
*   **☽ Mood Board**: Log your daily mood using a color palette, view a dynamic mood heatmap calendar, and track your mood history over time.
*   **✎ Journal**: A private space to reflect on your daily acts of kindness, tag how they made you feel, and look back on "On This Day" memories.
*   **✿ Affirmations**: Get daily affirmations, save your favorites, and set gentle intentions for your day.
*   **❋ Kind Wall**: A safe, anonymous space to share kind thoughts and read what others are sharing. No profiles, no pressure.
*   **◌ Zen Hub**: A collection of gentle, interactive mini-games designed to help you relax (e.g., Petal Drift, Breathing Bloom, Light Finder, Starlight Whisper).
*   **◌ The Release**: A private, temporary space to write out your thoughts and watch them dissolve—letting go of what you no longer need to hold onto.
*   **🎧 Ambient Sounds**: Built-in relaxing audio (Piano, Crystal Chimes) to accompany your stay.
*   **🌗 Vibe Selection**: Choose from varied color themes that adapt to both light and dark modes.

## Tech Stack 🛠️

This project is built using:
*   **HTML5**
*   **CSS3** (Custom properties for theming, modern layout techniques)
*   **Vanilla JavaScript** (No heavy frontend frameworks)

## Running Locally 🚀

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/daisyspaceco.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd daisyspaceco
   ```
3. **Serve the files:**
   Since it uses standard HTML/CSS/JS, you can simply open `index.html` in your browser. Alternatively, for a better development experience, use a local server like Live Server (VS Code extension) or Python's `http.server`:
   ```bash
   # Using Python 3
   python -m http.server 8000
   ```
   Then, navigate to `http://localhost:8000` in your web browser.
