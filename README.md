# daisyspaceco 🌼

> *wherever you are, you are welcome here*

**daisyspaceco** is a gentle corner of the internet — a fully anonymous, open-source web app for daily kindness, mindful reflection, and a little softness in your day. No accounts. No pressure. Just you.

🌐 **Live site:** [shxivya.github.io/daisyspaceco](https://shxivya.github.io/daisyspaceco)

---

## ✨ Features

### ✦ Gentle Acts
Get a daily prompt for a small act of kindness — for yourself or someone else. Track your day streak and total acts done, and build your own kindness chain over time.

### ☽ Mood Board
Log your daily mood through a soft colour palette. View your month as a heatmap calendar, track the last 14 days on a sparkline, and browse your mood history by week, month, or year.

### ✎ Journal
A private space to reflect. Write entries, tag how you felt, pin your favourites, and look back on memories from the same day in past years. Export your journal as a PDF anytime.

### ✿ Affirmations
Receive a gentle affirmation each day, save the ones that resonate, and set a quiet intention for the day ahead.

### ❋ Kind Wall
An anonymous, shared space to leave kind thoughts for strangers. Posts are stored in a real database so everyone who visits can read and react to what others have shared — no names, no profiles, ever.

### ◌ Zen Hub
Four gentle interactive experiences to help you slow down:
- 🌸 **Petal Drift** — interact with soft floating petals
- 🌬️ **Breathing Bloom** — follow a gentle breath rhythm
- 🕯️ **Light Finder** — uncover hidden words with a soft glow
- 🌌 **Starlight Whisper** — connect stars and watch them bloom

### ◌ The Release
Write out whatever you're holding. Watch it dissolve. Let it go.

### 🎧 Ambient Sounds
Built-in Piano and Crystal Chimes to accompany your stay.

### 🌗 Vibe & Theme
Choose from several colour themes, each working in both light and dark mode.

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML5, CSS3, Vanilla JavaScript |

| Fonts | Google Fonts (Playfair Display, Lato, Caveat, Cormorant Garamond) |

No frameworks. No build tools. Just the web. 🌿

## 🚀 Running Locally

```bash
# Clone the repo
git clone https://github.com/shxivya/daisyspaceco.git
cd daisyspaceco

# Serve with Python
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

Or use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code.

> **Note:** The Kind Wall requires a Supabase project to work. See [`js/wall.js`](js/wall.js) and replace `SUPABASE_URL` and `SUPABASE_ANON_KEY` with your own credentials.
